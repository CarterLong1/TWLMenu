Nintendo DS Titles

For games run with nds-bootstrap from the system SD card, TWiLight Menu++ has a menu where you can select cheats. The codes are in the Action Replay format, and they're packed into a usrcheat.dat file, which can be made using R4CCE. We recommend using the cheat database from DeadSkullzJr: cutt.ly/DSJCheats

Place the usrcheat.dat file in the sd:/_nds/TWiLightMenu/extras folder, then select cheats from the per-game settings menu.
