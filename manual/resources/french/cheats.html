Titres Nintendo DS

Pour les jeux exécutés avec nds-bootstrap de la carte SD du système, TWiLight Menu++ possède un menu où vous pouvez sélectionner les triches. Les triches sont au format Action Replay et sont réunies dans le fichier usrcheat.dat, qui peut être réalisé en utilisant R4CEE. Nous recommandons d'utiliser la base de données de triches de DeadSkullzJr: cutt.ly/DSJCheats

Placer le fichier usrcheat.dat dans le dossier sd:/_nds/TWiLightMenu/extras puis sélectionnez les triches dans les paramètres de chaque jeu.
